# GPC_Masterclass

## Windows
```
git clone https://github.com/vassvik/GPC_Masterclass.git
cd GPC_Masterclass
git submodule update --init --recursive
cd Odin
build.bat 1
cd ..
build.bat
```

## Linux

Install glfw from your favourite package manager, and it should hopefully link properly.
